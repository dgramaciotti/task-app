# Task app

Basic task (or todo) app to show some coding principles. 

FE stack: react + MUI + React query
BE: stack: Express + prisma

## Get started

### Running the FE

1. cd frontend
2. nvm use
3. npm install
4. npm run dev

### Running the BE

1. cd server
2. docker compose up
3. api should be running on port 3000
